# TaskFlow Web App

A static task management web page with login, create-account, and task CRUD behavior.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Add this folder's contents to the repository root and push to `main`.
3. In GitHub, open **Settings** > **Pages**.
4. Set **Build and deployment** to **Deploy from a branch**.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait for GitHub Pages to finish publishing.

## Notes

- No build step is required.
- Open `index.html` locally to test before publishing.
- The included `.nojekyll` file keeps GitHub Pages from processing the site with Jekyll.
